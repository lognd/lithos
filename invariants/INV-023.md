`docs/spec/regolith/13-invariants.md`; that file is the normative
source.

Mechanism: `check_route` treats regions as first-class owned entities;
placement, routing, features, and memory partitions all pass the same
region-intersection check, with overlap-where-permitted as an explicit
join declaration.

Argument summary: every spatial/resource occupancy is an entity
creation or modification, and all of those pass the ownership checker
-- there is no occupancy channel outside the entity DB.
