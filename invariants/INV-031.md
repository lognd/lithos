Checked mirror of INV-31 (Shipped sheets are legible by
construction) in `docs/spec/regolith/13-invariants.md`; that file is
the normative source.

Mechanism: `assert_ship_ready` shares ONE geometry home
(`measure_text_width_mm`/`wrap_to_width`/`fit_text` and
`DimensionGeometry`) with the renderer, so the audit's clip/overlap
check measures the EXACT same bbox the renderer is about to draw.
`DrawingsBackend.produce` calls it for every configured drawing spec
BEFORE any file for that subject is rendered; a failing rule returns
`Err(BackendError)`, propagating through `ship`'s all-backends-Ok gate.

Argument summary: soundness reduces to "the audit measures the
renderer's actual output geometry" composed with "a `Result::Err` from
any backend refuses the whole ship" -- a sheet that would clip or
overlap can only ship if both independently-computed geometries agree
it does not, which for pure functions over identical inputs is the
same computation twice.

Anchor (primary enforcing site): `python/regolith/backends/drawings/audit.py :: assert_ship_ready`
