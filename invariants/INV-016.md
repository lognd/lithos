`docs/spec/regolith/13-invariants.md`; that file is the normative
source.

Mechanism: `run_converter_check` builds the converter graph and checks
it for within-domain combinational cycles; every converter port kind
samples pre-instant and applies post-instant (a delta by type, not by
analysis).

Argument summary: any cross-boundary cycle contains a converter, and
every converter contains a delta, so no zero-delay cycle exists;
within-domain combinational cycles are separately rejected.
