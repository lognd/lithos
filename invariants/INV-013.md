Checked mirror of INV-13 (No dead uppers) in
`docs/spec/regolith/13-invariants.md`; that file is the normative
source.

Mechanism: `conformance_obligation` builds a `<upper> conforms <lower>`
claim for every impl/extern/import binding the compiler discovers; the
obligation exists by construction wherever both levels are authored.

Argument summary: there is no code path that registers a
lower-level realization without an upper binding except authoring
*only* the lower level, where the contract is authored there --
nothing is shadowed because nothing else exists.

Anchor (primary enforcing site): `crates/regolith-lower/src/claims/conformance.rs :: conformance_obligation`
