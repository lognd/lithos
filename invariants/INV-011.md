Checked mirror of INV-11 (Monomorphization totality) in
`docs/spec/regolith/13-invariants.md`; that file is the normative
source.

Mechanism: `monomorphize` expands every generic declaration over its
distinct use-site instantiations before L3 checks; a use site whose
arity mismatches its declaration is an un-expandable point (E0504) and
a generic referenced nowhere is a dead generic (E0503).

Argument summary: expansion is exhaustive by construction; variants
are externally-chosen so no point is skippable.

Anchor (primary enforcing site): `crates/regolith-lower/src/checks.rs :: monomorphize`
