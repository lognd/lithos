Checked mirror of INV-29 (Rule totality) in
`docs/spec/regolith/13-invariants.md`; that file is the normative
source.

Mechanism: attachment is union with qualified-name collision as an
error (E0602, `regolith-lower/src/rules.rs::check_rule_packs`); the
one evaluator classifies each attached rule per its D-E cases,
recording every deferral case (unmodeled domain, realized-tier domain
with no entities, filter tails) by name on `eval` -- never a silent
skip. `select_domain_entities` is the domain-resolution step that
decides between "evaluate" and "defer, naming why."

Argument summary: the evaluation function is total over attached
rules by construction -- its match arms are exactly {verdict,
deferral, definition error} and none is empty-bodied.

Anchor (primary enforcing site): `crates/regolith-lower/src/rule_engine.rs :: select_domain_entities`
